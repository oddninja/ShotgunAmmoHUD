# Changelog

## 1.0.0 — initial public release
Shows how many shells are loaded in the shotgun you're holding, as a native line in the game's
bottom-centre control-tip panel (e.g. "Shells : [1 / 2]"). The line is a clone of one of the
game's own control-tip text objects, so it matches the in-game font, size, colour and alignment,
and sits one line above the rest of the tips.
